# Chrome扩展 - YoutubeCN

# 注意：此项目非免VPN工具

现为半成品

## 功能：

- Youtube位置列表增加“中国”选项（部分页无效）

- 推送功能将关闭（位置在中国）

- Youtube搜索结果将跳转至[google.cn](https://google.cn/)（位置在中国）
